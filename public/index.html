Welcome to My-Koa

A simple and powerful Koa framework

A simple user Demo shows here!

Quick Start:

Here is an example of how to use My-Koa:


import Koa from 'koa';
import Router from 'koa-router';

const app = new Koa();
const router = new Router();

// Define routes
router.get('/', ctx => {
  ctx.body = 'Hello, My-Koa!';
});

app.use(router.routes())
   .use(router.allowedMethods());

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});

Enjoy building with My-Koa!
